### ⚠️ Issue Naming Rule

`[<PREFIX>] <Description>` 의 양식을 준수하되, Prefix는 협업하며 맞춰가기로 한다.
또한 Prefix는 대문자를 사용한다.

```
[FEAT] 회원가입 구현
[DEBUG] MainActivity 버그 수정
[STYLE] 폰트 변경
```
